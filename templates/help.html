AI Guide

🚀 How to Use the Crop Prediction System

  • Select crop, season, and state
  • Enter area and farming inputs
  • Click Predict to see yield

📊 Data Used

  • Crop data (production & area)
  • Weather data (temperature, rainfall, humidity)
  • Farming inputs (fertilizer, pesticide)

🌾 Input Field Details

  • Crop: Name of the cultivated crop
  • Season: Kharif, Rabi, Whole Year, etc.
  • State: Indian state for cultivation
  • Area: Total land in hectares
  • Fertilizer: Amount in kilograms
  • Pesticide: Amount in liters
  • Temperature: Average seasonal temperature (°C)
  • Rainfall: Annual rainfall (mm)
  • Humidity: Average seasonal humidity

📝 Important Note

Soil values like N, P, K, and pH are removed to keep the system simple and farmer-friendly.
